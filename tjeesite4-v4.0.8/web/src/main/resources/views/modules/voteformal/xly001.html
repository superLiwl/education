简要事迹

惠州日报讯 （记者杨 熠）龙舟水虽已结束，但新一轮强降雨又将来袭。据市气象台昨日预报，23日夜间至26日，我市将迎来新一轮强降水，并伴有8级以上雷雨大风等强对流天气。此次降水过程持续时间较长，各方须加强防御，市民出行应随身携带雨具、避让易积水路段。暴雨红色预警信号生效期间，停止户外作业和活动，中小学幼儿园停课。

据悉，今年龙舟水期间（5月21日～6月20日），全省平均雨量383.7毫米，较常年偏多2成，为2009年以来最强；全省平均日照时数100.7小时，较常年偏少近3成。“本次强降水我省主要落区在粤北，可能与6月10日～14日受灾严重区域有重叠，加上我省周边的江西、湖南、广西和福建也有大范围持续性强降水，上游江河水位将明显上升。”有关气象专家分析说。
